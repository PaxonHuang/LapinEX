# LapinEX

## 介绍

LapinEX 是一个基于 [Hexo](https://hexo.io/) 的静态博客系统，使用 [Next](https://github.com/theme-next/hexo-theme-next) 主题，并集成了 [Valine](https://valine.js.org/) 评论系统。

## 使用

### 安装

1. 安装 Hexo

```bash
npm install -g hexo-cli
```

2. 初始化博客

```bash
hexo init LapinEXBlog
cd LapinEXBlog
npm install
```
